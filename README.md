# 🚀 Professional Task Manager

A sophisticated, full-stack task management application combining the power of Next.js for a modern, responsive frontend and Django REST Framework for a robust backend.

---

## 🌐 Live Preview

👉 **Try it here:**  
🔗 https://client-three-alpha-62.vercel.app  

Experience the app in real-time with full functionality including task creation, filtering, and Kanban board interaction.

---

## 🌟 Features Overview

### Core Functionality

#### **Task Management**
- Complete CRUD operations
- Kanban-style board (To Do, In Progress, Completed)
- Priority system (High, Medium, Low)

#### **Intelligent Due Date Tracking**
- Flexible due dates
- Smart urgency color indicators:
  - 🔴 Overdue  
  - 🟠 Due Today  
  - 🟡 Due Soon  
  - 🟢 On Track  

#### **Advanced Search & Filtering**
- Real-time search
- Sorting by latest, due date, priority
- Persistent session state

#### **Comprehensive Task Information**
- Created & updated timestamps
- Detailed descriptions
- Visual badges for quick scanning
